AJAX

Данная технология позволяет получать данные с сервера без перезагрузки страниц при этом вычислительная нагрузка по отображению информации ложиться на JS, а соответственно на клиентскую машину. С данной технологией можно работать из чистого JS, а так-же из Jquerry.

1. Работа через JS. В данном подходе для формиромания асинхронного запроса используется объект XMLHTTPRequest (XHR). Для того чтобы совершить асинхронный запрос данного объекта следует выполнить следующие стадии:

	1. Подготовка запроса. Для этого следует воспользоваться методом Open(). Данный метод принимает 2 обязательных параметра: название http метода, а так-же адресс по которому будет провоизводиться запрос.

	2. Отправка запроса. Для этого используется медот Send(). В данный метод может быть передана строка содержащая данные отправляемыми методом Post.
В некоторых случаях может понадобиться перед отправкой запроса добавить некоторые заголовки. Для этого можно воспользоваться методом setRequestHeader(). Если необходимо прервать запрос до его окончания можно воспользоваться методом abort().


Свойства объекта XMLHTTPRequest

readyState - содержит в себе число характеризующее в себе этап выполнения запроса.

onreadystatechange - событие которое генерируется при изменении значения свойства readyState. Обычно интерес составляет изменение статуса на 4 (Ответ получен).

status - вданное свойство помещается код HTTP ответа.

statusText - содержит описание HTTP ответа. Обычно интересует статус 200 или статус-текст "ОК".

Данные HTTP ответа помещаються в свойства responseText и responseXML. ResponseText - содержит данные в виде строки. ResponseXML - в случае использования формата XML данные обрабатываються и содержаться в виде DOM модели XML документа.
